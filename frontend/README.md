# 📌 Collab Design - Collaborate, Design, Succeed

**Collab Design** is a desktop-focused web application built to make students' lives easier by simplifying software documentation tasks.  
It unifies diagramming, requirement management, team communication, and discussion features — all in one collaborative platform.

---

## 🌟 Key Features

✅ **Automated & Manual UML Diagramming**  
Create various popular UML diagrams manually or generate select diagrams automatically to save time and maintain clarity.

✅ **Requirement Management**  
Document and organize project requirements using a structured form. Download forms for offline use or future reference.

✅ **Version Control**  
Track changes to project artifacts, view past versions, and restore them as needed. Prevents data loss and promotes accountability.

✅ **Built-in Team Chat**  
Communicate in real time with your team, ensuring everyone stays connected and on track.

✅ **Discussion Forum**  
Engage with peers, share insights, ask questions, and exchange ideas to support your academic projects.

## 📝 Problem Statement

Many existing tools offer limited functionality, forcing users to switch between different apps for diagrams, requirement management, and communication — which disrupts productivity and focus.  
**Collab Design** addresses this by integrating these tools into a single, easy-to-use platform.

## 📦 Installation & 🚀 Running the Project

To run **Collab Design** locally:

```bash

# Navigate into the project directory
cd collab-design

# Install dependencies
npm install

# Start the development server
npm run dev

```
